# MVP
This is the Repository for the MVP program that will run on the Raspberry Pi. It is divded into different libraries

## Subsections

Listed by alphabetical order

Section | Description
--------|------------
ExternalGPIO | Takes care of any hardware that is connected to the Raspberry Pi
Modem | Takes care of SMS (Receiving and Sending)
MVP | Core of the Program, dispatches everything
ReminderControl | Takes care of any remind part
RunControl | Update, log, closing, state of the program
STT | Speech To Text (Obvious)
TTS | Text To Speech (Obvious too)
